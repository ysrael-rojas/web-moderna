La Web Moderna.

HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de páginas web. Funciona a través de etiquetas y atributos que definen elementos como texto, imágenes, enlaces y tablas, proporcionando la base para integrar estilos (CSS) e interactividad (JavaScript).

CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para diseñar y personalizar la apariencia de páginas web creadas con HTML. Permite separar el contenido de la presentación, facilitando el mantenimiento y la reutilización de estilos en diferentes documentos.

Javascript

JavaScript es un lenguaje de programación ampliamente utilizado para desarrollar páginas web interactivas. Permite manipular el DOM para modificar dinámicamente el contenido de las páginas, validar formularios y agregar animaciones.

HTTP

HTTP (Hypertext Transfer Protocol) es el protocolo que permite la comunicación entre un cliente (como un navegador web) y un servidor, facilitando la transferencia de datos como páginas web, imágenes y videos.
