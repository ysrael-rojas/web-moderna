La Web Moderna.

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de páginas web. Funciona a través de etiquetas y atributos que definen elementos como texto, imágenes, enlaces y tablas, proporcionando la base para integrar estilos (CSS) e interactividad (JavaScript). Es esencial para el desarrollo web y se mantiene como estándar desde su introducción en la década de 1990

CSS (Cascading Style Sheets) es un lenguaje utilizado para diseñar y personalizar la apariencia de páginas web creadas con HTML. Permite separar el contenido de la presentación, facilitando el mantenimiento y la reutilización de estilos en diferentes documentos. Con CSS se puede controlar colores, fuentes, márgenes, alineaciones, y mucho más, lo que lo convierte en una herramienta esencial para desarrollar interfaces atractivas y funcionales

JavaScript es un lenguaje de programación ampliamente utilizado para desarrollar páginas web interactivas. Permite manipular el DOM para modificar dinámicamente el contenido de las páginas, validar formularios y agregar animaciones. Además, gracias a frameworks como React o Angular, se emplea para crear aplicaciones web completas

HTTP (Hypertext Transfer Protocol) es el protocolo que permite la comunicación entre un cliente (como un navegador web) y un servidor, facilitando la transferencia de datos como páginas web, imágenes y videos. Funciona mediante un modelo de solicitud-respuesta, donde el cliente envía solicitudes y el servidor devuelve los recursos solicitados. Es una tecnología clave para la navegación en la web
